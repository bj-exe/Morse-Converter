# This project is a web-based Morse Code Converter. It allows users to:

Convert Text → Morse and Morse → Text.

Play Morse code as audio tones with customizable waveform, WPM, and frequency.

Copy results to the clipboard.

Toggle between light and dark themes.

# Features
Bidirectional conversion: Toggle between text and Morse instantly.

Playback: Hear Morse code as beeps with adjustable waveform (sine, square, triangle, sawtooth).

Speed control: Set words per minute (WPM) for playback speed.

Frequency control: Adjust tone frequency in Hz.

Theme toggle: Switch between light and dark mode.

Clipboard copy: Copy converted output with one click.

# Settings
Accessible via the Settings modal:

Alphabet: Currently supports Latin (A–Z, 0–9).

Waveform: Choose sine, square, triangle, or sawtooth.

WPM: Words per minute (default: 20).

Frequency: Tone frequency in Hz (default: 550).

# Usage
Enter text in the Text box (left side).

Click Convert → Morse code appears in the right box.

Click Toggle → Switch direction (Morse → Text).

Use Play Output → Hear Morse as tones.

Use Copy Output → Copy result to clipboard.
